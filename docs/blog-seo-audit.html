Google SEO Requirements Checklist

Each blog post should meet these Google SEO standards for optimal ranking:

1. Meta Title (Title Tag)

50-60 characters, primary keyword near start, brand name optional at end. Example: "Climbing Mount Kilimanjaro: Complete 2026 Guide | Snow Africa"

2. Meta Description

150-160 characters, include primary keyword, clear benefit/CTA. Example: "Plan your Kilimanjaro climb with our expert guide. Routes, costs, best seasons & packing lists. Book with Tanzania's trusted operator."

3. URL Slug

Short, descriptive, hyphenated, include keyword. Current slugs are good but verify they match content focus.

4. H1 Heading

One per page, matches search intent, contains primary keyword. Should match or be similar to meta title.

5. Content Quality (E-E-A-T)

Experience, Expertise, Authoritativeness, Trustworthiness. Show first-hand experience, cite sources, author credentials.

6. Internal Linking

Link to related posts, service pages, and booking forms. Helps distribute page authority and keeps users engaged.

7. Image Optimization

Descriptive alt text, compressed file sizes, relevant file names. Featured image for social sharing (1200x630px).

8. Schema Markup

Article/BlogPosting schema with author, datePublished, image. Enables rich snippets in search results.
